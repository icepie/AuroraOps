连接能力
自动注册设备、建立 TCP 长连接、定期心跳并同步 CPU、内存、硬盘、网卡等资产；页面与 Electron 均通过本地 HTTP 控制 Go Agent

采集预览

用于确认本机实际上报的物理资产结构，并对比服务端已有资产，区分新增、更新、保持不变和仅服务端存在的数据。

点击“预览资产”后显示本机采集结果

采集诊断

展示 CPU、内存、磁盘、网卡、主板等采集阶段的执行结果，便于定位为什么资产为空。

点击“预览资产”后显示采集诊断结果
